# Autenticação 

Aplicação backend para gerenciar acesso de usuários a uma plataforma.

## Funcionalidades

- Criar conta
- Fazer login
- Excluir conta 
- Editar conta
- Fazer autenticação através do JWT


### Tecnologias utilizadas

- Bcrypt
- Dotenv
- Express
- JSON Web Token
- Mongoose
- Node
- Nodemon
- Passport
- Passport HTTP Bearer
- Passport Local

### Instalando as dependências

```
$npm install
```

### Iniciar API

```
$npm start
```

### Home

```
localhost:3000
```
